# Omnisense AI Bridge for Unity

Omnisense is an autonomous AI agent bridge and editor suite powered by the Model Context Protocol (MCP) for Unity 3D.

## Features

- **AI Chat & Orchestration**: Context-aware AI assistant running directly inside the Unity Editor (`Window > Omnisense > AI Chat`).
- **AI Image Generation & Editor**: Generate textures and edit image assets directly in Unity (`Window > Omnisense > Image Generator` / `Image Editor`).
- **3D Model Generation**: Prompt-based 3D mesh and asset generation (`Window > Omnisense > 3D Model Generator`).
- **MCP Server Protocol**: Full Model Context Protocol (MCP) support for tool execution, scene manipulation, and component inspection.

## Getting Started

1. Open Unity and navigate to `Window > Omnisense > AI Chat`.
2. Configure your API provider settings (OpenAI, Anthropic, Gemini, or self-hosted endpoint) in the **Settings** tab.
3. Start interacting with the AI agent to automate scene creation, component editing, or asset generation!

## Menu Shortcuts

- `Window > Omnisense > AI Chat`: Open main AI Assistant window.
- `Window > Omnisense > Image Generator`: Open AI Texture & Image Generator.
- `Window > Omnisense > Image Editor`: Open Image Editing Suite.
- `Window > Omnisense > 3D Model Generator`: Open 3D Model Generation Workspace.
